# Purposeful Living After 60

Website for **[purposefullivingafter60.com](https://purposefullivingafter60.com)** — a professional, inspirational blog focused on health, retirement, and digital income for a fuller second act.

## Overview

Static HTML/CSS/JS site, hosted on **Netlify**, domain registered through **Namecheap**.

- Tagline: *"Feel strong every day"*
- Three core categories: **Health**, **Retirement**, **Digital Income**

## Structure

| File / Folder | Purpose |
|---|---|
| `index.html` | Homepage |
| `articles.html` | The Journal (article index) |
| `article-*.html` | Individual articles |
| `about.html` | About page |
| `contact.html` | Contact page |
| `privacy.html` | Privacy policy |
| `thanks.html` | Newsletter/contact thank-you page |
| `style.css` | Site styles |
| `app.js` | Site JavaScript |
| `assets/` | Images and favicon |
| `sitemap.xml`, `robots.txt` | SEO |

## Deployment

The site can be deployed to Netlify. To enable **auto-deploy on every push**, connect this
repository to the Netlify site in Netlify → Site settings → Build & deploy → link repository.

- **Build command:** *(none — static site)*
- **Publish directory:** `.` (repository root)
